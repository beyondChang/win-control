# 电视时光 - OpenClaw api 调用说明
适配 OpenClaw 调用，读取后，用户可以通过对话调用，本地接口统一请求 `127.0.0.1:26679`。

---

## 1. 电视时光_通用搜索视频
- **触发场景**：用户说「搜索xxx」「帮我找xxx」「我想看xxx」「电视时光搜xxx」
- **逻辑**：自动按当前平台搜索；无平台时默认用腾讯视频搜索
- **接口**
  - URL：`http://127.0.0.1:26679/command`
  - 方法：`POST`
- **参数**
  - `payload`：搜索关键词（如：海绵宝宝、甄嬛传、新闻联播）
- **请求体**
  ```json
  {
    "action": "searchVideo",
    "payload": "{{payload}}"
  }
  ```

---

## 2. 电视时光_换台
- **触发场景**：「换到xxx台」「我要看xxx台」「切换xxx频道」「看xxx卫视」「换台」
- **别名支持**：中央1台=CCTV1、中央5台=CCTV5、湖南台=湖南卫视等
- **逻辑**：未打开电视视图则自动打开，再切换频道
- **接口**
  - URL：`http://127.0.0.1:26679/command`
  - 方法：`POST`
- **参数**
  - `payload`：必须从**频道枚举**中选择
- **请求体**
  ```json
  {
    "action": "jumpToChannel",
    "payload": "{{payload}}"
  }
  ```
- **支持频道**
  CCTV1~CCTV17、CCTV4K/8K、CGTN 系列、各省市卫视、中国教育电视台1频道、新疆卫视等。

---

## 3. 电视时光_打开平台
- **触发场景**：「打开B站」「去爱奇艺」「打开腾讯视频」「看抖音」「开电视」
- **接口**
  - URL：`http://127.0.0.1:26679/command`
  - 方法：`POST`
- **参数**
  - `payload`：平台枚举值
- **枚举映射**
  - `tengxun`：腾讯视频
  - `bili`：B站/哔哩哔哩
  - `aiqiyi`：爱奇艺
  - `youku`：优酷
  - `mgtv`：芒果TV
  - `douyin`：抖音
  - `tv`：电视直播
  - `yangku`：央视片库
- **请求体**
  ```json
  {
    "action": "openView",
    "payload": "{{payload}}"
  }
  ```

### 统一说明
- 所有接口均为 `application/json`
- 本地服务地址：`127.0.0.1:26679`
- 仅需按用户意图匹配工具名，OpenClaw 会自动填充参数并调用
